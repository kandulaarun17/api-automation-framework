# Employee API Automation Framework
A REST Assured-based API Automation Framework developed using Java, TestNG, and Maven to automate CRUD operations on an Employee Management API. 
The framework follows industry-standard design principles and supports data-driven testing, API chaining, reporting, logging, and schema validation.

## Tech Stack
- Java
- REST Assured
- TestNG
- Maven
- Jackson
- JSON Schema Validation
- Apache POI (Excel)
- Extent Reports
- Log4j2
- Git & GitHub
- JSON Server

## Features
- API automation for POST, GET, PATCH, and DELETE operations
- API Chaining
- Data-Driven Testing using Excel
- POJO Serialization & Deserialization
- Request & Response Specifications
- JSON Schema Validation
- Negative Test Scenarios
- Extent HTML Reports
- Log4j2 Logging
- Reusable Utilities (Config, Excel, Reporting, Logging)
- Page Object Model style project structure for maintainability
